# NaviDate - 커플 데이트 플래너

> 둘이 함께 고르고, 매일 새로운 데이트를 발견하는 스마트 추천 서비스

---

## 서비스 소개

NaviDate는 커플 두 사람의 대화와 선택을 학습하여 점점 더 정확한 데이트 코스를 추천해주는 웹 서비스입니다.

---

## 3가지 핵심 차별점

### 1. 커플 싱크 (Couple Sync) - 함께 고르기

```
┌─────────────────────────────────────────────────────────────┐
│                    💕 Couple Sync                           │
│                                                             │
│   ┌─────────────┐              ┌─────────────┐             │
│   │  👧 수진    │              │  👦 민수    │              │
│   │             │              │             │              │
│   │  [성수 카페] │              │  [성수 카페] │             │
│   │   ← ❌  ✅ →│              │   ← ❌  ✅ →│             │
│   │             │              │             │              │
│   │  상대 선택:  │              │  상대 선택:  │             │
│   │  ✅ 좋아요!  │              │  ⏳ 선택 중  │             │
│   └─────────────┘              └─────────────┘             │
│                                                             │
│              둘 다 ✅ → 🎉 데이트 코스에 추가!              │
└─────────────────────────────────────────────────────────────┘
```

**특징:**
- 실시간으로 상대방 선택 확인 가능 (투명한 소통)
- 둘 다 좋아요한 장소만 데이트 코스에 추가
- "어디 가?" "몰라" 대화 종결

---

### 2. 스마트 학습 추천 - 선택이 다음 추천에 반영

```
┌─────────────────────────────────────────────────────────────┐
│                  🧠 AI 학습 시스템                          │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 📊 커플 취향 프로필                                  │   │
│  │                                                      │   │
│  │  대화 분석                                           │   │
│  │  ├─ "오늘 파스타 먹고 싶다" → 양식 +1               │   │
│  │  └─ "요즘 전시회 가고 싶었어" → 전시 +1             │   │
│  │                                                      │   │
│  │  선택 학습                                           │   │
│  │  ├─ 성수동 카페 ✅ 3회 → 성수 지역 선호도 ↑        │   │
│  │  ├─ 시끄러운 술집 ❌ 2회 → 조용한 분위기 선호      │   │
│  │  └─ 주말 브런치 ✅ 5회 → 브런치 카테고리 ↑        │   │
│  │                                                      │   │
│  │  데이트 후 평가                                      │   │
│  │  └─ ⭐⭐⭐⭐⭐ "분위기 최고" → 해당 장소 유형 ↑    │   │
│  └─────────────────────────────────────────────────────┘   │
│                           ↓                                 │
│            🎯 다음 추천에 자동 반영!                        │
└─────────────────────────────────────────────────────────────┘
```

**학습 요소:**
| 입력 | 학습 내용 | 반영 |
|------|-----------|------|
| 채팅 대화 | 언급된 음식, 장소, 활동 키워드 | 관심 카테고리 파악 |
| 스와이프 선택 | 좋아요/패스한 장소 패턴 | 선호 지역, 분위기, 가격대 |
| 데이트 후 평가 | 별점, 한줄평, 재방문 의사 | 추천 정확도 향상 |
| 방문 빈도 | 자주 가는 카테고리/지역 | 루틴 패턴 파악 |

```

├─────────────────────────────────────────────────────────────┤
│                                                             │
│  🔥 오늘의 핫플                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 🆕 성수동 새로 오픈한 브런치 카페                    │   │
│  │ 📸 인스타 요즘 핫한 포토존                           │   │
│  │ 🎬 이번 주 개봉 영화 추천                            │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  🎉 오늘만 특별한                                           │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 🎪 성수 팝업스토어 (1/26~2/10)                       │   │
│  │ 🍽️ 레스토랑 위크 할인 (~1/31)                       │   │
│  │ 🎨 무료 전시회 마지막 주                             │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  🌤️ 날씨 맞춤                                              │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 오늘 맑음 3°C → 실내 데이트 추천                     │   │
│  │ 추천: 방탈출 카페, 보드게임 카페, 실내 클라이밍      │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  💕 우리 커플 맞춤 (학습 기반)                              │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 민수 & 수진님이 좋아할 것 같은 곳                    │   │
│  │ → 성수 신상 파스타집 (양식 선호 + 성수 자주 방문)   │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

**자동 수집 정보:**
- 🆕 신규 오픈 맛집/카페
- 🎪 진행 중인 팝업스토어/전시회
- 🎬 최신 영화/공연 정보
- 🏷️ 할인/이벤트 정보
- 🌤️ 실시간 날씨 기반 추천
- 📍 지역별 핫플레이스 트렌드

## 기술 스택

| 영역 | 기술 | 용도 |
|------|------|------|
| 프론트엔드 | React, TypeScript, Tailwind | UI/UX |
| 백엔드 | Node.js, Express | API 서버 |
| 데이터베이스 | MongoDB | 사용자, 커플, 데이트 기록 |
| 실시간 통신 | Socket.io | 커플 싱크, 채팅 |
| AI 추천 | Claude API | 대화 분석, 맞춤 추천 |
| 지도/장소 | 카카오맵 API | 장소 검색, 지도 표시 |
| 데일리 정보 | 웹 크롤링 + 외부 API | 트렌드, 이벤트 수집 |
| 스케줄러 | Node-cron | 매일 자동 업데이트 |
| 배포 | Vercel + Railway | 프론트/백엔드 호스팅 |

## 데이터 모델

### User (사용자)
```javascript
{
  id: string,
  email: string,
  nickname: string,
  profileImage: string,
  coupleId: string | null,
  createdAt: Date
}
```

### Couple (커플)
```javascript
{
  id: string,
  coupleCode: string,
  users: [userId, userId],
  region: string[],
  transport: string,
  createdAt: Date
}
```

### CouplePreference (학습된 선호도)
```javascript
{
  coupleId: string,

  // 카테고리별 선호도 점수 (학습으로 업데이트)
  categoryScores: {
    food: { korean: 5, western: 8, japanese: 3, ... },
    activity: { active: 4, calm: 7, ... },
    mood: { romantic: 6, hip: 8, cozy: 5, ... }
  },

  // 지역 선호도
  regionScores: {
    "성수": 10, "홍대": 6, "강남": 4, ...
  },

  // 가격대 선호
  budgetPreference: "medium",

  // 최근 관심 키워드 (채팅 분석)
  recentInterests: ["파스타", "전시회", "브런치"],

  updatedAt: Date
}
```

### DailyContent (데일리 큐레이션)
```javascript
{
  date: Date,

  hotPlaces: [{
    name: string,
    category: string,
    reason: string,  // "신규 오픈", "SNS 화제" 등
    location: object,
    source: string
  }],

  events: [{
    title: string,
    period: { start: Date, end: Date },
    location: string,
    type: "popup" | "exhibition" | "festival" | "sale"
  }],

  weather: {
    condition: string,
    temperature: number,
    recommendation: string
  },

  updatedAt: Date
}
```

### DateHistory (데이트 기록)
```javascript
{
  id: string,
  coupleId: string,
  date: Date,
  places: [{
    placeId: string,
    name: string,
    category: string,
    rating: number,
    tags: string[],  // "맛있어요", "분위기좋아요" 등
    revisit: boolean
  }],
  memo: string,
  photos: string[],

  // 학습에 반영됨
  feedbackApplied: boolean
}
```
